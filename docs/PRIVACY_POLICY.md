# Política de Privacidade — Central 190

**Versão:** 1.0.0 RC — 17 de junho de 2026.

O Central 190 não exige cadastro, não possui telemetria, não envia o save para servidores próprios e não vende dados pessoais.

Os dados de carreira ficam no armazenamento local do navegador. A localização do jogador só é solicitada após ação explícita no botão correspondente, é arredondada para uma região aproximada e permanece no aparelho. Ela pode ser removida na Central de Lançamento.

O mapa real pode solicitar imagens cartográficas a um provedor externo de tiles. O jogo não armazena esses tiles em seu Service Worker. O fallback tático funciona sem conexão.

O jogador pode exportar, importar, restaurar backup ou reiniciar o save pelas configurações.
