# Fase 31 — Fila contínua e ícones cinematográficos SP

Esta fase corrige a sensação de que a próxima ligação não cai após o despacho.

## Fluxo novo

1. O jogador despacha a primeira ocorrência.
2. O rádio operacional inicia.
3. Se não houver chamada em espera, a próxima ligação programada é antecipada.
4. A tela mostra uma faixa: "Próxima ligação em Xs".
5. Quando a ligação entra, a fila viva é atualizada com toque/aviso.
